# Commerce Coaching Websites

Three commerce coaching institute websites, each with a distinct, hand-crafted identity (no shared template). Static HTML/CSS/vanilla-JS — no build step.

| Site | Identity | Folder |
|------|----------|--------|
| **Soni Classes Commerce** | Established academic prestige — navy + gold, editorial serif | [`soni-classes/`](soni-classes/) |
| **Reliable Classes** | Results-driven, data-forward — emerald + amber, results ledger & charts | [`reliable-classes/`](reliable-classes/) |
| **Parab Commerce Classes** | Founder-led mentorship — maroon + marigold, warm notebook motifs | [`parab-classes/`](parab-classes/) |

Each site has `index.html`, `courses.html`, `contact.html`, a shared `css/styles.css`, and `js/main.js`.

> Phone, email and address are placeholders — replace before going live. Result figures and testimonials are realistic demo content.

## Run locally

```sh
python3 -m http.server 8000
# open http://localhost:8000
```
