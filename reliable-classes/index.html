Method

How the result is built

The numbers in Table 1 are not luck. They come from one fixed system that every batch runs through, every week, with nothing left to chance.

  1. Week-1 diagnostic test

    Before any teaching, every new student sits a written diagnostic across accountancy, maths and English.

    The score places the student in one of three pace groups so revision starts from what they actually know — not from page one for everyone.

  2. Syllabus split into 18 units

    Each subject’s board syllabus is broken into tracked units, each with a hard completion deadline.

    A wall chart marks every unit taught / revised / tested, so a student and parent can see exactly where the batch stands in week 19 of 30.

  3. Saturday test + rank sheet

    Every Saturday the batch sits a full timed paper on the week’s units, marked to board standard.

    Scores are posted on a named rank sheet by Monday. 36 tests a year means no student can quietly fall behind unnoticed.

  4. Monthly report to parents

    Once a month, each family gets a written report: rank trend, units behind, and attendance.

    It is paired with a 1:1 doubt log — every question a student raises is recorded and checked off when resolved.

The same figure as Table 1, drawn so the trend is plain. The dashed line marks the Maharashtra HSC Commerce state average for reference.

  • The gap to the state average is roughly 6–9 points every year, which is the difference the method in the section above is built to create.
